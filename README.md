# Generative AI with LangChain & LangGraph

Hands-on notebooks covering the **LangChain** and **LangGraph** ecosystem — from first GenAI apps to agents, embeddings, and vector stores.

## Topics

**Building apps & chatbots**
- `BUILDING_SIMPLE_GENAI_APP` / `BUILD_SIMPLE_GENAI_APP_USING_OLLAMA` — first GenAI apps (hosted + local Ollama models)
- `BUILDING_CHATBOT_LANGCHAIN` — conversational chatbot with memory
- `CHATTRANSLATOR_LANGCHAIN_LCEL` — translation chain using LangChain Expression Language
- `LANGGRAPH` — stateful, graph-based agent workflows

**Agents**
- `SQL_AGENT` / `SQL_AGENT_LANGCHAIN` — natural language → SQL over a database
- `ANALYTICS_AGENT` — data analytics agent

**Data ingestion & retrieval**
- `DATAINGESTIONLANGCHAIN_DOCUMENT_LOADERS` — document loaders
- Text splitters: character, recursive character, HTML
- Embeddings: OpenAI, Hugging Face
- Vector stores: **FAISS**, **Chroma DB**

**Foundations**
- `PYDNATICS` — Pydantic models for structured LLM output

## Tech stack

Python · LangChain · LangGraph · OpenAI · Ollama · FAISS · ChromaDB · Hugging Face
